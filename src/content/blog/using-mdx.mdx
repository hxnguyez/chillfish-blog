This theme comes pre-installed and configured with the [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) integration. The specific configurations are already set in the `astro.config.mjs` file. If you prefer not to use MDX, you can simply remove the integration from the configuration file to disable it.

## Why MDX?

MDX is a special Markdown variant that supports embedded JavaScript and JSX syntax. This allows you to [mix JavaScript and UI components into your Markdown content](https://docs.astro.build/en/guides/markdown-content/#mdx-features), which is useful for creating interactive charts, alert boxes, and other dynamic features.

If you already have content created with MDX, this integration makes it easy to migrate to Astro.
