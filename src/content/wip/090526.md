## Optimizing my workflow  
Today, I adjust my wsl2 to be more convenient with me:
- Download qemu and bonus alias to open by one command (I will push blog about that in Tool tommorow)
- Editing alias for upload git file (this blog) by editing .bashrc file and add this bash script in the bottom:
```bash
blog() {
    local BLOG_PATH="$HOME/chillfish/chillfish-blog"
    
    if [ ! -d "$BLOG_PATH" ]; then
        echo "[-] Error: Path $BLOG_PATH not found."
        return 1
    fi

    cd "$BLOG_PATH" || return 1
    
    if [ -z "$(git status --porcelain)" ]; then
        echo "[!] Nothing to update!"
        return 0
    fi
    
    local msg="${1:-update blog}"
    echo "[+] Deploying with message: '$msg'..."
    
    git add .
    git commit -m "$msg"
    git push
    
    echo "[+] Done. Connection closed."
}
```
So when I use blog command with "text commit" (or not) it will use combination of three commands git add, git commit and git push
